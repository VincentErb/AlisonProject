The Alison Project

A state-of-the-art sound recognition solution designed to help people suffering from hearing impairement all over the world.

Alison is an open source project developped with Python for Raspberry Pi. Calibrate the device to listen for some of the sounds of your house using our cross-platform mobile app. Alison will inform you of the occurence of these sounds in a visual way.
